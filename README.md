# 🌌 AI Galaxy

AI Galaxy is a platform that helps users explore the universe of Artificial Intelligence tools in one place.
It provides information about AI tools, AI agents, and modern AI technologies used in different fields such as coding, image generation, productivity, and education.

## 🚀 Features

* Collection of popular AI tools
* Categories like AI Chatbots, AI Coding Assistants, AI Image Generators
* Simple interface to explore tools
* Useful for students, developers, and researchers

## 🧠 AI Categories

* AI Chatbots
* AI Coding Tools
* AI Image Generators
* AI Video Tools
* AI Productivity Tools
* AI Research Tools

## 🎯 Goal

The goal of AI Galaxy is to create a central hub where anyone can discover and learn about the rapidly growing ecosystem of AI technologies.
